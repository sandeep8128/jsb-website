# 🚀 JSB Website — Figma to React Implementation

A modern, responsive, and visually appealing frontend website developed by converting a professional **Figma UI/UX design into a fully functional React.js application**.

## 🌐 Live Demo

live demo link - https://jsb-website.vercel.app


## 🎨 Design Source

UI/UX Design:
**Designed in Figma & converted into React.js**

The goal of this project was to accurately implement the Figma design into a pixel-perfect and interactive frontend interface.
